# Ejercicio Test Case — TC-01

## Funcionalidad
Cambio de contraseña del usuario autenticado

---

## Caso de prueba

**ID:** TC-01  
**Título:** Verificar cambio de contraseña con datos válidos  
**Precondición:** El usuario se encuentra logueado y en la pantalla de cambio de contraseña.

**Pasos:**
1. Introducir la contraseña actual válida
2. Introducir una nueva contraseña válida
3. Pulsar el botón de guardar cambios

**Resultado esperado:**  
El sistema cambia la contraseña correctamente y muestra un mensaje de confirmación al usuario.
