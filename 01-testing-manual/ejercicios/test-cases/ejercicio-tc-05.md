# Ejercicio Test Case — TC-05

## Funcionalidad
Mensajes de error mostrados al usuario en el login

---

## Caso de prueba

**ID:** TC-05  
**Título:** Mostrar mensaje de error con credenciales incorrectas  
**Precondición:** El usuario se encuentra en la página de login.

**Pasos:**
1. Introducir un nombre de usuario válido
2. Introducir una contraseña incorrecta
3. Pulsar el botón "Login"

**Resultado esperado:**  
El sistema no inicia sesión y muestra un mensaje de error indicando que las credenciales son incorrectas.
