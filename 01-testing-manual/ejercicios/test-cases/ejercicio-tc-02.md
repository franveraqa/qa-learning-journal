# Ejercicio Test Case — TC-02

## Funcionalidad
Edición del perfil de usuario

---

## Caso de prueba

**ID:** TC-02  
**Título:** Verificar edición del perfil con datos válidos  
**Precondición:** El usuario se encuentra autenticado y en la pantalla de edición del perfil de usuario.

**Pasos:**
1. Modificar el campo "Nombre" con caracteres válidos
2. Modificar el campo "Correo" con un email válido
3. Pulsar el botón de guardar cambios

**Resultado esperado:**  
El sistema guarda correctamente los cambios realizados y muestra un mensaje de confirmación al usuario.
