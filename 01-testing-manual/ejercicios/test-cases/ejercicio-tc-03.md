# Ejercicio Test Case — TC-03

## Funcionalidad
Cierre de sesión (Logout) del usuario

---

## Caso de prueba

**ID:** TC-03  
**Título:** Verificar cierre de sesión del usuario  
**Precondición:** El usuario se encuentra autenticado y en una página donde está disponible la opción de cerrar sesión.

**Pasos:**
1. Pulsar el botón "Cerrar sesión"
2. Esperar la respuesta del sistema

**Resultado esperado:**  
El sistema cierra la sesión del usuario y lo redirige a la pantalla de inicio de sesión.
