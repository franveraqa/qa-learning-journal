# Ejercicio Bug Report — BUG-03

## Bug

**ID:** BUG-03  
**Título:** El botón de login no responde al pulsarlo  
**Precondición:** El usuario se encuentra en la página de login.

**Pasos para reproducir:**
1. Introducir un nombre de usuario válido
2. Introducir una contraseña válida
3. Pulsar el botón "Login"

**Resultado esperado:**  
El sistema autentica al usuario y permite el acceso a la aplicación.

**Resultado real:**  
El botón de login no responde al pulsarlo y no se muestra ningún mensaje al usuario.

**Severidad:** Alta  
**Prioridad:** Alta

---

## Justificación
La severidad es alta porque el usuario no puede acceder al sistema, bloqueando una funcionalidad principal.  
La prioridad es alta porque impide el uso de la aplicación.
