# Test Plan — Funcionalidad Registro de Usuario

## Objetivo
Verificar que la funcionalidad de registro permite crear nuevos usuarios correctamente, gestionando validaciones y errores de forma adecuada.

---

## Alcance (In Scope)
Las pruebas incluirán:

- Registro con datos válidos
- Validación de campos obligatorios
- Validación de formato de email
- Validación de contraseña
- Mensajes de error mostrados al usuario

---

## Fuera de alcance (Out of Scope)
Quedan fuera de este plan:

- Confirmación por email
- Seguridad avanzada
- Rendimiento y carga
- Integraciones externas

---

## Tipos de pruebas
- Pruebas funcionales
- Smoke testing
- Regression testing

---

## Entorno de pruebas
- Navegador: Google Chrome  
- Entorno: Test  
- Dispositivo: Escritorio  

---

## Criterios de entrada
- Funcionalidad de registro desarrollada
- Entorno disponible
- Formulario accesible

---

## Criterios de salida
- Casos de prueba ejecutados
- Bugs críticos y altos reportados
- Sin bloqueos que impidan continuar las pruebas

---

## Notas finales
Este Test Plan sirve como guía para validar la funcionalidad de registro de usuarios y podrá actualizarse en futuras versiones.
