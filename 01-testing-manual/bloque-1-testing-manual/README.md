# Bloque 1 — Fundamentos y Testing Manual

Este bloque forma parte de mi formación como **QA Junior** y cubre los fundamentos del testing manual.

## Contenido del bloque
En este bloque se han trabajado las siguientes áreas:

- Diseño de casos de prueba para la funcionalidad de login
- Ejecución de pruebas funcionales
- Reporte de bugs de forma profesional
- Análisis de severidad y prioridad
- Creación de un Test Plan básico

## Estructura
El bloque se organiza de la siguiente manera:

- **test-cases/**  
  Casos de prueba diseñados para validar la funcionalidad de login.

- **bug-reports/**  
  Bugs documentados con pasos para reproducir, resultado esperado, resultado real, severidad y prioridad.

- **test-plan/**  
  Test Plan básico que define el alcance, tipos de prueba y criterios de entrada y salida.

## Objetivo
Demostrar conocimientos sólidos de testing manual y documentación QA, aplicables a un entorno real de trabajo.
